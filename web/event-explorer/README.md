# Event Explorer UI

Заглушка для будущего web-интерфейса.

MVP-экран должен включать:

- фильтр по периоду;
- фильтр по ИБ;
- фильтр по пользователю;
- фильтр по имени события;
- фильтр по объекту метаданных;
- фильтр по сеансу, соединению, транзакции и приложению;
- быстрый поиск по представлению данных, комментарию и raw-представлению;
- таблицу событий;
- карточку события;
- экспорт результата.
