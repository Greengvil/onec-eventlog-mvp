# AltLog Explorer / Event Explorer

Заглушка для будущего web-интерфейса AltLog Explorer.

Экран Event Explorer на текущем MVP-этапе должен включать:

- фильтр по периоду;
- фильтр по ИБ;
- фильтр по пользователю;
- фильтр по имени события;
- фильтр по объекту метаданных;
- фильтр по сеансу, соединению, транзакции и приложению;
- быстрый поиск по представлению данных, комментарию и raw-представлению;
- таблицу событий;
- карточку события;
- экспорт результата.
